2024年度企业招聘政策总纲

一、招聘流程
1. 职位发布：HR部门根据各部门人员需求，在招聘平台和公司官网发布职位信息
2. 简历筛选：HR在收到简历后3-5个工作日内完成初步筛选
3. 笔试环节：技术类岗位需参加在线笔试，考试时长90分钟，涵盖专业知识和逻辑测试
4. 面试安排：通过笔试的候选人将进入面试环节，一般安排2-3轮面试
   - 第一轮：技术面试（60分钟）
   - 第二轮：综合面试（45分钟）
   - 第三轮（如需）：高管面试（30分钟）
5. 录用通知：面试通过后5个工作日内发出录用通知书

二、面试准备材料
1. 个人身份证原件及复印件
2. 学历学位证书原件及复印件
3. 相关职业资格证书（如有）
4. 近期一寸免冠照片2张
5. 上一家公司的离职证明（社招）

三、入职政策与试用期
1. 新员工入职需签订劳动合同，合同期限一般为3年
2. 试用期规定：
   - 合同期限3年及以上：试用期不超过6个月
   - 合同期限1年以上不满3年：试用期不超过2个月
   - 合同期限不满1年：试用期不超过1个月
3. 试用期薪资不低于转正后薪资的80%
4. 试用期考核：每月进行一次，连续两次考核不合格者不予转正

四、薪资福利政策
1. 薪资结构：基本工资 + 绩效工资 + 年终奖金
   - 基本工资：根据岗位级别和地区标准确定
   - 绩效工资：每月绩效考核结果决定，占比20%-40%
   - 年终奖金：根据年度绩效和公司经营情况发放，一般为1-3个月工资
2. 五险一金：
   - 养老保险、医疗保险、失业保险、工伤保险、生育保险
   - 住房公积金（按最高比例缴存）
3. 带薪年假：
   - 工作满1年不满10年：5个工作日
   - 工作满10年不满20年：10个工作日
   - 工作满20年及以上：15个工作日
4. 其他福利：
   - 补充商业医疗保险
   - 年度健康体检
   - 餐补及交通补贴
   - 节日福利金
   - 员工培训发展计划

五、内部推荐政策
1. 推荐人资格：公司在职正式员工均可参与内部推荐
2. 推荐流程：通过公司内部推荐系统提交被推荐人简历
3. 推荐奖励：
   - 一般岗位：被推荐人通过试用期后，推荐人获得3000元奖金
   - 高级岗位：被推荐人通过试用期后，推荐人获得8000元奖金
   - 管理岗位：被推荐人通过试用期后，推荐人获得15000元奖金
4. 推荐人不得参与被推荐人的面试评审

六、校园招聘特别政策
1. 招聘时间：每年9月-11月（秋季）和3月-5月（春季）
2. 招聘对象：应届毕业生及毕业2年内的往届生
3. 培养计划：新入职校招生将进入"启航计划"，为期6个月的系统培训
4. 薪酬：校招生起薪根据学历层次和院校等级确定
5. 住宿补贴：异地校招生入职前3个月享受每月1500元住宿补贴

七、注意事项
1. 本政策由人力资源部负责解释和修订
2. 如遇政策调整，以最新版本为准
3. 涉及个人薪资福利的具体细节，以最终录用通知书和劳动合同为准
4. 如有疑问，请联系HR部门邮箱：hr@company.com
